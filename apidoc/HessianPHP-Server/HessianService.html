Class HessianService

Description
Description | Vars (details) | Methods (details)

<BLOCKQUOTE> Enables a common PHP object to be published as a Hessian compatible web-service via a POST request.

It wraps a PHP object that executes the real methods passed by the HessianService class.

Example usage:

  1.  $wrapper = &new HessianService();<BR>
  2.  $wrapper->registerObject(new ServiceObject());<BR>
  3.  $wrapper->service();<BR>

Then clients can perform method calls to the url of the service, ex. http://localhost/hessian/testservice.php

		service (line 198)
	
 
	
	

Publishes the service, check incoming calls and routes them to the wrapped object.
  This method uses streams to retrieve raw POST bytes and a  HessianPHP.HessianParser and HessianPHP.HessianWriter  to execute the call and send results back to the client.


As defined in Hessian 1.0 spec, the service requires POST to execute. It is advised not to call
  any other php code that writes to the default screen output (echo, print, etc.) as it can corrupt  the reply.
